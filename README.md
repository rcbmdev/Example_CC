# Example_CC

Projeto de exemplo para estudo e demonstração do **Claude Code** — a CLI oficial da Anthropic para o Claude.

## Sobre o Projeto

Este repositório foi criado como ambiente de aprendizado para explorar as funcionalidades do Claude Code, incluindo:

- Geração e edição de código assistida por IA
- Integração com repositórios Git
- Uso de ferramentas e agentes especializados
- Automação de tarefas de desenvolvimento

## Pré-requisitos

- [Node.js](https://nodejs.org/) 18+
- [Claude Code CLI](https://github.com/anthropics/claude-code) instalado globalmente

```bash
npm install -g @anthropic-ai/claude-code
```

## Como Usar

1. Clone o repositório:

```bash
git clone https://github.com/rcbmdev/Example_CC.git
cd Example_CC
```

2. Inicie o Claude Code no diretório:

```bash
claude
```
